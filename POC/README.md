# Portal Academico POC

Prueba de concepto de un portal web estatico para visualizar seguimiento academico por estudiante, alertas tempranas y prioridades de acompanamiento.

## Contenido

- `index.html`: estructura principal del portal.
- `styles.css`: identidad visual y comportamiento responsive.
- `script.js`: datos demostrativos, filtros y render dinamico.

## Publicacion web

El repositorio queda preparado para desplegar esta carpeta en GitHub Pages usando el workflow definido en `.github/workflows/deploy-pages.yml`.

## Nota

Los datos del portal son demostrativos y reflejan la estructura del archivo de seguimiento academico encontrado en el workspace.
